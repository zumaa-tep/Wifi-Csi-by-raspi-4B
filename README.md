# Wifi-Csi-by-raspi-4B
ใช้ raspi 4B ที่ติดตั้ง firmware NEXMON ในการดักจับสัณญาณ wifi csi โดยนำมาใช้ในการจำแนกพฤติกรรมของมนุษย์โดยใช้โมเดล SVM และ LSTM ในการจำแนกพฤติกรรม
